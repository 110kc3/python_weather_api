  • What this application does

    This is application takes specified stations data with help of airly API and displays it. You can specify and add airly stations or your own (configured before)

    Adding public station

    You can add public station from airly service by specifying its coordinates or you can put City name that you want your station from and application will find you the closest station.

    Custom station configuration

    The instruction on how to configure your custom station running on Raspberry Pi can be found here: Git link

    Adding custom station

    If you want to connect your custom station to this application you will need its ip address and port that station listens to.
